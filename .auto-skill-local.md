## 🧠 Expert Knowledge: Story-First Content Curation
- **Core Pattern**: [Person] -> [Counter-intuitive Hook] -> [Story Context] -> [Verified Stats].
- **Research Hook Check**: If you can't summarize the story in one surprisingly catchy sentence (e.g., "The family that taught Taiwan credit"), research deeper.
- **Structure Checklist**: Always include `30-second overview`, `Callouts (📝/⚠️/💡)`, and `Stat Blocks` for visual density.
- **Sync Note**: On Windows local, use PowerShell `Copy-Item` if `sync.sh` (bash) is unavailable.
